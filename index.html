📱 Android — Add to Home Screen
1. Open your Netlify URL in Chrome
2. Tap ⋮ menu → "Add to Home screen"
3. Tap Add — opens fullscreen like an app ✓
To use custom icon: Long-press shortcut → Edit → tap icon → Gallery → select downloaded PNG
🖥 Windows Edge — Install as App
1. Open Netlify URL in Microsoft Edge
2. Click ⋯ menu → Apps → Install this site as an app
3. Or click the install icon in the address bar (screen + arrow)
4. Opens in its own window, no browser bar ✓
